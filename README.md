# Online Course Management System (MERN)

## Project Overview
This is a full-stack MERN application developed to manage courses in an organization.
It supports complete CRUD (Create, Read, Update, Delete) operations with persistent
storage using MongoDB.

-----

## Technology Stack
- **Frontend:** React (Vite), Bootstrap
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **API Type:** REST (JSON)

----

## Features
- Add new courses
- View all available courses
- Update existing course details
- Delete courses
- Data stored persistently in MongoDB
- Responsive UI using Bootstrap
